# Gaming Portal with User Authentication

A web-based gaming platform developed using Flask, MySQL, HTML, CSS, and JavaScript. The application provides user registration and login authentication, allowing users to access and play Tic-Tac-Toe and Rock-Paper-Scissors games through a centralized dashboard.

## Features

* User Registration
* User Login Authentication
* MySQL Database Integration
* Tic-Tac-Toe Game
* Rock-Paper-Scissors Game
* Interactive User Interface

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS
* JavaScript

## Project Structure

templates/

* loginpage.html
* registerpage.html
* gamepg.html
* index_tic.html
* index_rock.html

static/

* CSS files
* JavaScript files
* Images

## How to Run

1. Clone the repository
2. Install Flask and MySQL Connector
3. Configure MySQL database
4. Run:

python app.py

5. Open:

http://127.0.0.1:5000
